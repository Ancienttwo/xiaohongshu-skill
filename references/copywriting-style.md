# Copywriting Style

Use this reference when drafting note titles, body text, and cover copy in `04-content-calendar.md` and `05-daily-ops.md`. Every note must feel like a friend sharing a discovery, not a brand delivering a pitch.

## Voice and Tone

The default voice is **girlfriend chat** (闺蜜安利): warm, excited, slightly exaggerated, always personal.

Rules:
- Write every sentence as spoken language, never written-essay language.
- Use first and second person constantly: 我、你、姐妹们、宝宝们、咱、家人们.
- Express genuine emotion: surprise, excitement, regret, relief. Flat descriptions kill engagement.
- Never sound like a brand, a textbook, or a press release.
- Mild exaggeration is expected. Deadpan understatement is not.

## Sentence Rules

- Maximum 15 characters per sentence (one breath, one idea).
- One sentence per line. Blank lines between groups of 2-3 sentences.
- No long compound sentences. Split them.
- End key sentences with exclamation marks or rhetorical questions.

Example rhythm:

```
姐妹们听我说！！

这个粉底液真的绝了✨
上脸就是奶油肌
持妆12小时不暗沉

之前用过XX完全不行
斑驳卡粉到怀疑人生😭

换了这个之后
素颜出门都有人夸
```

## Emoji Usage

Emoji on Xiaohongshu is **structural**, not decorative. It replaces punctuation, creates visual rhythm, and signals content type.

### Placement rules

| Position | Purpose | Example |
|---|---|---|
| Title | Eye-catch + content signal | 3步搞定底妆✨零毛孔 |
| Paragraph opener | Section marker | ✅第一步：选对色号 |
| Mid-sentence | Replace a noun or emotion | 用了之后皮肤变好到💥 |
| Line-end | Emotional punctuation | 真的会谢😭😭😭 |
| Between sections | Visual divider | ➖➖➖➖➖➖ or 🔻🔻🔻 |

### Category-specific emoji

| Category | Core emoji set |
|---|---|
| Universal fire | 🔥 ⚡ ✨ 💯 🎯 💎 ⭐ 🌟 |
| Food | 🍜 🍰 ☕ 🍳 🥗 😋 |
| Beauty/skincare | 💄 💋 🧴 💆 🪞 ✨ |
| Fashion | 👗 👖 👠 🧥 💅 🎀 |
| Fitness | 💪 🏋️ 🔥 ⚡ 🏃 |
| Home/life | 🏠 🛋️ 🧹 🌿 ☀️ |
| Emotion markers | 😭 🥺 😍 🤯 💕 ❤️ |

### Anti-patterns

- Do not use more than 3 emoji in a single sentence.
- Do not stack 5+ identical emoji in a row.
- Do not use emoji that contradict the tone (skull emoji in a positive review).
- Do not use emoji as the only content of a line.

## Power Words (爆款词)

Rotate these naturally. Do not stuff multiple power words into one sentence.

| Category | Words |
|---|---|
| Urgency | 快给我冲、赶紧去囤、必码、一定试试、别再错过 |
| Superlative | 绝绝子、YYDS、好用到哭、吹爆、神器、天花板 |
| Authority | 教科书般、压箱底、吐血整理、手把手、揭秘 |
| Identity | 家人们、姐妹们、打工人、学生党、手残党、小白必看 |
| Emotion | 破防了、治愈、高级感、沉浸式、被夸爆、笑不活了 |
| Collection | 建议收藏、划重点、合集、清单、宝藏 |

## Title Formula

Target: 20 characters or fewer. Must contain one keyword and one emotional hook.

### Six proven patterns

| Pattern | Template | Example |
|---|---|---|
| Number | [数字]+[动作]+[结果] | 3步搞定奶油底妆✨ |
| Question | [痛点]？+[方案] | 毛孔粗大怎么办？这瓶救了我 |
| Contrast | [反差A]+[反差B] | 月薪3千穿出3万的感觉 |
| Reverse | [违反常识的主张] | 不运动也能瘦10斤 |
| Proof | 我用[时间]+[达成结果] | 我用7天把痘痘全消了 |
| Warning | 别再[错误行为]！ | 别再这样洗脸了！毁皮肤 |

### Title checklist

- [ ] 20 characters or fewer
- [ ] Contains one search keyword
- [ ] Contains one emotional or outcome angle
- [ ] Only one dominant promise, not multiple stacked claims
- [ ] At least one emoji if the category allows it

## Body Structure

### Standard flow (种草/教程)

```
[Hook: 1-2 sentences, pain point or result teaser]

[Section 1: problem statement or context]
- short bullet
- short bullet

[Section 2: solution or method, with proof]
- step or finding
- step or finding

[Section 3: personal experience or result]

[CTA: soft interaction prompt]

[Hashtags: 5-10 tags]
```

### Hook patterns

| Type | Example |
|---|---|
| Pain point | 你是不是也... / 每次...都想哭 |
| Result teaser | 用了这个之后再也回不去了 |
| Social proof | 被三个姐妹追问链接 |
| Curiosity gap | 千万别在下午三点打开这个APP |
| Confession | 后悔没早点知道这个方法 |

### CTA patterns (soft, never pushy)

- 姐妹们用过吗？评论区聊聊～
- 还想看什么类型的？告诉我！
- 觉得有用记得收藏💕下次找得到
- 你们还有什么好用的推荐吗？

Do not write: 点赞关注收藏转发！ (too aggressive, triggers user resistance)

## Hashtag Rules

- Place 5-10 hashtags at the end of the note body.
- Include 2-3 broad category tags and 3-5 specific long-tail tags.
- Format: #关键词# or #关键词 (platform accepts both).
- Do not invent hashtags that nobody searches for.

Example set for a skincare note:

```
#护肤 #敏感肌 #屏障修复 #平价护肤品推荐 #油痘肌护肤 #学生党护肤
```

## Quality Check Before Publishing

Every note must pass these checks:

1. Does every sentence read like spoken Chinese, not written Chinese?
2. Are sentences under 15 characters with frequent line breaks?
3. Do emoji serve a structural purpose (not just scattered randomly)?
4. Is there exactly one clear hook in the first 2 lines?
5. Is there a soft CTA near the end?
6. Are 5-10 relevant hashtags at the bottom?
7. Does the note avoid banned words and aggressive sales language?

If a note fails 2 or more checks, revise before adding to the calendar.
